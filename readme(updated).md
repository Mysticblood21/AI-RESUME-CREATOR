# AI Resume Builder

1. Install dependencies
```bash
pip install flask requests reportlab python-dotenv pillow
```

## Run Locally

```bash
python app.py
```

Open: http://localhost:5000

## Run online with ngrok

1. Start Flask
```bash
python app.py
```

2. Start ngrok
```bash
ngrok http 5000
```

3. Open Forwarding website that shows up in terminal
